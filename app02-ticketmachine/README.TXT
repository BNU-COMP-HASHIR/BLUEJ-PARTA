PROJECT TITLE: TICKET MACHINE

PURPOSE OF PROJECT: A ticket machine that lets you buy train tickets for three stations ( Aylesbury, High Wycombe, Amersham) after you have inserted atleast the minimum fare required for each destination.

VERSION or DATE: 20th October 2020

HOW TO START THIS PROJECT: Start the BlueJ file. create a ticket machine from the ticket class. To make things easier and save time, you can call the method to create all the three tickets the machine has to offer. 
You can then insert specific coins through methods in the machine and select the required buy ticket function to get the ticket you want.
